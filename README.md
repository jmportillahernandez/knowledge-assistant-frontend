# Knowledge Assistant Frontend

Frontend en React para un sistema RAG empresarial basado en Qdrant.

## Features
- Upload de documentos PDF
- Listado de documentos
- Reindexado y eliminación
- Chat con IA
- Filtros por metadata
- Visualización de fuentes

## Tech Stack
- React
- Vite
- Axios

## Run
npm install
npm run dev
